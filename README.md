# AESCSFv2

AESCSF v2 Evidence Tracker
Overview

The AESCSF v2 Evidence Tracker is a single-file HTML application designed to assist organisations in documenting evidence, tracking implementation status, and managing remediation timelines for the Australian Energy Sector Cyber Security Framework (AESCSF) Version 2 practices.

This tool is designed to be simple, portable, and require no server or database. All data is stored locally in the browser and can be exported/imported via JSON.

Features
Record evidence against each AESCSF practice
Track implementation status (Partial, Largely, Fully)
Assign owners to practices
Record notes and remediation actions
Set target dates and review dates
Timeline view for remediation scheduling
Export data to JSON
Import data from JSON
Local browser storage (offline capable)
Single HTML file (no installation required)
How to Use
Download the HTML file.
Open the file in a web browser (Chrome, Edge, or Firefox recommended).
Enter evidence, notes, owners, and target dates for each practice.
Click Save to store data in the browser.
Use Export JSON to back up or transfer your assessment.
Use Import JSON to restore previously saved data.
Use the Timeline tab to view remediation schedules and overdue items.
Status Definitions
Status	Meaning
Partial	Some controls implemented but gaps remain
Largely	Mostly implemented with minor gaps
Fully	Fully implemented and evidenced
Data Storage

Data is stored in:

Browser Local Storage
Exported JSON files for backup and transfer

Important: Clearing browser data will remove saved assessments unless exported.

Intended Use

This tool is intended for:

AESCSF self-assessments
Cyber security maturity assessments
Evidence tracking for audits
Remediation planning and scheduling
Compliance documentation
Limitations
Runs locally only (no multi-user support)
No authentication
No database backend
Designed for single assessor use
